The goal of this project is to predict stock prices for upcoming years using the Facebook Prophet model. It includes a Python web app built with Streamlit, offering an intuitive interface for users to interact with the model and view predictions for different time horizons.

The Smart Bank application allows users to register an account, securely log in, and access their dashboard. Users can transfer money to others using usernames, and the bank admin can manage funds through an admin page. The application is built with HTML, CSS, JavaScript, REST API, and MongoDB.

Technologies Used: HTML, CSS, JavaScript, REST API, MongoDB
